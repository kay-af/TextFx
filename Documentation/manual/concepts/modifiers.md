# Modifiers

Modifiers determine how a character is modified based on the other settings of the group. A group has multiple modifiers which you can enable individually.

## Types

The following section describes the available modifier types.

### Position Modifier

Controls the local position of the characters.

### Rotation Modifier

Controls the local rotation of characters with the configured pivot of the group.

### Scale Modifier

Controls the local scale of characters with the configured pivot of the group.

### Color Modifier

Controls the color of the characters.

### Opacity Modifier

Controls the opacity of the characters.

> [!NOTE]
> The magnitude of each of these properties are determined using the **strength** and the effect range of the group.
