Namespace TextFx

Classes

BaseModifier

The base class of all modifiers responsible for applying an effect on a character.

ColorModifier

Controls the color of characters.

Mask

A simple fractional mask that defines a range within the [0-1] space.

Used to determine which characters should be affected by a group based on their position within the text. Values are fractional [0-1] and does not depend on the actual width.

ModifierGroup

A group encapsulating characters and provides an interface to control various modifiers and settings related to the characters in the group.

OpacityModifier

Controls the opacity of characters.

PlateauCurve

A special type of curve that starts rising from 0 to 1 following a profile (Rising profile), is 1 for a certain range, and then falls back to 0 following another profile (Falling profile).

The values used are fractional i.e, 0 is considered the starting point and 1, the ending point no matter the actual width of the range it is applied to.

PositionModifier

Controls the local translation of characters.

RotationModifier

Controls the local rotation of characters around the configured pivot.

ScaleModifier

Controls the local scale of characters around the configured pivot.

TextFxController

The main TextFx controller provides an interface to create text effects using modifier groups and exposes controls to drive animations.

This controller provides functionalities that support any TMPro.TMP_Text implementations.

TextFxUtils

Structs

ModificationContext

Context used internally to compute and pass modified character information across all groups and their modifiers sequentially. Each modifier applies its properties to this context, which accumulates the final transformation, color, and opacity for each character.
