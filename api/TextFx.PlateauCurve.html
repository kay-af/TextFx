A special type of curve that starts rising from 0 to 1 following a profile (Rising profile), is 1 for a certain range, and then falls back to 0 following another profile (Falling profile).

The values used are fractional i.e, 0 is considered the starting point and 1, the ending point no matter the actual width of the range it is applied to.
