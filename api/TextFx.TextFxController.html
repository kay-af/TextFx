Class TextFxController

Namespace

The main TextFx controller provides an interface to create text effects using modifier groups and exposes controls to drive animations.

This controller provides functionalities that support any TMPro.TMP_Text implementations. 

[ExecuteInEditMode]
[DisallowMultipleComponent]
[RequireComponent(typeof(TMP_Text))]
[AddComponentMenu("TextFx/TextFxController")]
public class TextFxController : MonoBehaviour

AutoUpdate

When enabled, the text mesh updates every frame. Disable it and call UpdateMesh() only on changes for performance. Effective only during play mode.

public bool AutoUpdate { get; set; }

Property Value

bool

Offset

The global fractional offset that is added to all the group offsets.

Offset is the starting fractional position of the range affected by the effects. Paired with extent of groups, it defines the range for which the group modifiers are applied. 

public float Offset { get; set; }

Strength

The global strength multiplier for all the effects.

The value must be in range [0,1]. The assigned value is clamped to this range. 

public float Strength { get; set; }
